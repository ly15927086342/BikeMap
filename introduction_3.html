建站技术

  1. ASP.NET及bootstrap

    整体框架用ASP.NET，页面布局用bootstrap，部署在腾讯云服务器IIS7

  2. 天地图 Javascript API

    大部分地图功能都用此接口，如调用底图，缩放，自定义控件等

  3. MySQL Database

    部署在阿里云服务器，含bicycledb和user两个数据库，包括point（节点）、polyline（弧段）、address（地名）、record（路线记录）、userinfo（用户信息）、word（留言板）这6个表。
